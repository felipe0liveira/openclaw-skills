# OpenClaw Skills

A collection of skills for [OpenClaw](https://openclaw.ai), available on [ClawHub](https://clawhub.ai/u/felipe0liveira).

## Available Skills

| Skill | Directory | Description |
|-------|-----------|-------------|
| gcloud | [`gcloud/`](gcloud/) | Manage Google Cloud Platform resources via the official gcloud CLI |

## Contributing

Contributions are welcome! To add a new skill or improve an existing one:

1. Fork this repository
2. Create a new branch for your changes
3. Add your skill in its own directory with a `SKILL.md` file
4. Submit a pull request

Each skill should follow the OpenClaw skill format with proper YAML frontmatter and clear instructions.
